# 📁 Código del Robot

Archivo principal del proyecto:

## 🤖 WRO-Revamp-2026.llsp3

Este archivo contiene el programa completo del robot desarrollado en **LEGO SPIKE Prime**.

### 🔧 Funcionalidades

* Movimiento autónomo
* Detección de colores
* Corrección de trayectoria
* Esquiva de obstáculos
* Control de estabilidad (yaw)

### ⚠️ Nota

Este archivo debe abrirse con el software oficial de **LEGO SPIKE Prime**.

No es directamente ejecutable como archivo `.py`, ya que utiliza el entorno visual de LEGO.
